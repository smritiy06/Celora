# 🚀 AI Learning Platform

An AI-powered learning platform that helps students study smarter through:

- 📚 Personalized Learning Roadmaps  
- 📝 AI-Generated Quizzes  
- 📊 Performance Analytics  
- 🎯 Weak Topic Detection  
- 🤖 AI Tutoring Assistance  
- 📈 Progress Tracking  

## ✨ Features

### Personalized Learning Roadmaps
Generate customized learning paths based on chosen subjects or topics.

### AI Quiz Generator
Create dynamic quizzes with multiple question types and difficulty levels.

### Performance Analytics
Track scores, learning consistency, progress, and overall performance.

### Weak Topic Detection
Identify weak areas using quiz results and learning behavior.

### AI Tutor Assistant
Get instant explanations, summaries, and doubt-solving support.

### Progress Tracking
Monitor completed topics, study goals, and learning milestones.

## 🛠 Tech Stack

**Frontend:**  
Next.js • TailwindCSS • Framer Motion • ShadCN UI

**Backend:**  
FastAPI • PostgreSQL • JWT Authentication • REST APIs

**AI / ML:**  
Scikit-learn • TensorFlow/PyTorch • Gemini API / OpenAI API

## 🎯 Goal

Build a modern educational platform that combines **AI, full-stack development, analytics, and personalized learning** to create a smarter study experience.
